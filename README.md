# Customer Service Request Analysis

## Project Overview
Customer Service Analysis is a data analysis project that explores customer support interactions to identify patterns, common issues, response quality, and opportunities to improve customer satisfaction.

## Tools Used
- Power BI
- Python
- Excel

## Files Included
- Dashboard images (.png)
- Analysis notebooks (.ipynb)
- Dataset (.csv)

## Key Insights
- Identified common complaint types
- Analyzed response time and closing time
- Compared complaints across different cities

## Conclusion
This project helps in improving customer service performance using data-driven insights.
